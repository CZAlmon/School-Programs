### Binary Search Tree for Games



##### Purpose: 

To store a list of computer games using binary search trees

Given a testfile, of form: GameName:	YearMade	ScoreGiven
the program will open and read in the testfile data, creating a Binary search tree for the list of games. From there the user will be able to insert new games, search existing games, list all games in the chosen order, and modify existing games user scores. 

The program does not have an option to save any changes made.
